# 14 - Selection

Text selection across renderables and scroll boxes.

## Concepts

- Selectable text elements with `~selectable:true`
- Custom selection colors with `~selection_bg` and `~selection_fg`
- Selection across multiple text elements
- Selection within scrollable content
- Cross-renderable selection spanning nested boxes
- Unicode text selection support
- Ctrl+click to extend selection

## Run

```bash
dune exec ./mosaic/examples/14-selection/main.exe
```
