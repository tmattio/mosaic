# 13 - Markdown

Markdown rendering with full CommonMark support.

## Concepts

- Markdown component with rich text formatting
- Headings, bold, italic, strikethrough, inline code
- Links and autolinks
- Ordered and unordered lists with nesting
- Fenced code blocks with syntax highlighting
- Blockquotes with nesting
- Tables and task lists
- Horizontal rules
- Scrollable content with scroll_box

## Run

```bash
dune exec ./mosaic/examples/13-markdown/main.exe
```
