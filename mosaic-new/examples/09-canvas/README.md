# 09 - Canvas

Procedural drawing surface for custom graphics.

## Concepts

- Off-screen canvas buffer
- Drawing primitives: plot, fill_rect, draw_box, draw_line
- Braille lines for sub-cell precision (2x4 resolution per cell)
- Animated drawing with frame updates
- Style and color control

## Run

```bash
dune exec ./mosaic/examples/09-canvas/main.exe
```
