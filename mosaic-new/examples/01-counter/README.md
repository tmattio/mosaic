# 01 - Counter

Minimal TEA example: model-view-update pattern with keyboard input.

## Concepts

- TEA structure: `init`, `update`, `view`, `subscriptions`
- Keyboard event handling with `Sub.on_key`
- Box and Text renderables
- Quitting with `quit ()`

## Run

```bash
dune exec ./mosaic/examples/01-counter/main.exe
```
