# 08 - Table

Rich data tables with columns, headers, and styling.

## Concepts

- Column definitions with headers
- Fixed and auto column widths
- Text alignment: left, center, right
- Box styles: Rounded, Heavy, Double, ASCII, Minimal
- Alternating row styles
- Cell-level styling

## Run

```bash
dune exec ./mosaic/examples/08-table/main.exe
```
